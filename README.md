# 🍽️ Zomato Restaurant Intelligence Dashboard

## 📌 Project Overview

An interactive Power BI dashboard built using the Zomato restaurant dataset containing 55,000+ restaurant records. The dashboard provides insights into restaurant distribution, cuisine popularity, customer ratings, online ordering trends, and market performance across locations.

## 🎯 Objectives

* Analyze restaurant performance across different locations.
* Identify the most popular cuisines.
* Understand customer rating patterns.
* Evaluate online ordering and table booking trends.
* Create an interactive business intelligence dashboard for decision-making.

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)

* Total Restaurants
* Average Rating
* Total Votes
* Average Cost for Two
* Table Booking Percentage

### Visualizations

* Top Cities by Restaurant Count
* Top 10 Cuisines
* Restaurant Rating Distribution
* Restaurant Density Map
* Most Popular Restaurants Table

### Interactive Filters

* Location
* Primary Cuisine
* Restaurant Type
* Rating Category
